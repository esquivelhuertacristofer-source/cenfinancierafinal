# MANUAL DE USUARIO
## Campaña Educativa Nacional - Plataforma de Educación Financiera

**Derechos Patrimoniales**: CEN - Centro de Educación Nacional  
**Diseñada e implementada por**: Luminar Studio

---

## 1. Introducción

### 1.1 ¿Qué es la Plataforma?
La **Campaña Educativa Nacional** es una plataforma educativa digital para la enseñanza de finanzas personales, diseñada para estudiantes de educación media y superior en México.

La plataforma es **modular, editable y adaptable** a cualquier branding o label empresarial, lo que permite personalizar la experiencia visual según las necesidades de cada organización.

### 1.2 Requisitos del Sistema
- Navegador web actualizado (Chrome, Firefox, Edge, Safari)
- Conexión a internet
- Cuenta de usuario registrada

---

## 2. Primeros Pasos

### 2.1 Registro e Inicio de Sesión
1. Accede a la plataforma desde el navegador
2. Haz clic en **"Iniciar Sesión"** en la esquina superior derecha
3. Si no tienes cuenta, el sistema te permitirá registrarte
4. Ingresa tu correo electrónico y contraseña
5. ¡Listo! Ya puedes acceder a todos los cursos

### 2.2 Navegación Principal
La barra de navegación contiene:
- **Inicio**: Página principal de la plataforma
- **Primeros Pasos**: Curso introductorio a las finanzas
- **Construyendo el Futuro**: Curso de planeación financiera
- **Planificación**: Estrategias de ahorro e inversión
- **Hora de Emprender**: Emprendedurismo y negocios

---

## 3. Funcionalidades del Estudiante

### 3.1 Cursos y Lecciones
- Explora los módulos de cada curso
- Cada curso se divide en **bloques** (lecciones)
- Lee el contenido educativo de cada bloque
- Visualiza juegos educativos interactivos (SecureGame)

### 3.2 Registro de Progreso
- Al finalizar cada bloque, haz clic en **"Finalizar este bloque"**
- El sistema registrará tu progreso automáticamente
- Verás una confirmación verde con marca de verificación
- Tu avance queda guardado en la base de datos

### 3.3 Seguimiento de Actividades
- El sistema registra el tiempo que pasas en cada actividad
- Cada minuto se registra automáticamente para auditoría

---

## 4. Funcionalidades del Docente

### 4.1 Panel Docente
Si tienes rol de docente, verás un botón **"Panel Docente"** en la barra de navegación.

Desde el dashboard puedes:
- Ver la lista de estudiantes registrados
- Monitorear el progreso de cada estudiante
- Visualizar estadísticas de completación de cursos

### 4.2 Gestión de Grupos
- Los docentes pueden gestionar grupos de estudiantes
- Asignar estudiantes a grupos específicos
- Revisar actividades por grupo

## 6. Juegos Educativos

### 6.1 ¿Qué son los SecureGames?
Los SecureGames son juegos educativos integrados en las lecciones:
- Se ejecutan en un entorno seguro (sandbox)
- Contenido interactivo para practicar conceptos
- Se cargan de forma diferida (lazy loading)

### 6.2 Cómo usar los juegos
1. Localiza el juego dentro de la lección
2. Interactúa con los elementos del juego
3. Practica los conceptos financieros aprendidos
4. No requieren instalación adicional

## 8. Solución de Problemas

### 8.1 No puedo iniciar sesión
- Verifica que tu correo esté registrado
- Confirma que la contraseña sea correcta
- Recupera tu contraseña si la olvidaste

### 8.2 No se guarda el progreso
- Asegúrate de tener conexión a internet
- Verifica que las cookies estén habilitadas
- Intenta cerrar sesión y volver a entrar

### 8.3 Los juegos no cargan
- Verifica tu conexión a internet
- Actualiza tu navegador
- Desactiva bloqueadores de publicidad

### 8.4 El panel docente no aparece
- Confirma que tu rol sea "teacher" en el sistema
- Contacta al administrador para cambiar tu rol

## 9. Glosario

| Término | Definición |
|---------|------------|
| **Progreso** | Registro que indica qué bloques has completado |
| **Perfil** | Información personal del usuario en la plataforma |
| **Rol** | Tipo de usuario: estudiante, docente o admin |
| **Bloque** | Una lección individual dentro de un curso |
| **Segmento** | Categoría principal de curso (ej: Primeros Pasos) |
| **Branding** | Personalización visual de la plataforma |
| **Modular** | Arquitectura que permite modificar componentes independientemente |
